# Insurance Analytics Dashboard | Power BI

## Project Overview

The Insurance Analytics Dashboard is an end-to-end business intelligence solution built using Power BI. The project provides a comprehensive view of insurance performance by analyzing customers, policies, premium revenue, claims, and trends over time.

The dashboard helps stakeholders monitor key KPIs, identify growth opportunities, understand customer behavior, and support data-driven decision making.

---

## Business Requirements

The objective of this project is to:

- Monitor overall business performance using KPIs.
- Analyze customer demographics and behavior.
- Evaluate policy performance across different policy types.
- Track premium and claim trends over time.
- Identify patterns and business opportunities.
- Support strategic and data-driven decision making.

## Key Insights

- Travel policies generated the highest premium revenue.
- Coverage amount significantly exceeds premium revenue.
- Monthly trends reveal seasonal fluctuations.
- Customer demographics influence premium contribution.
- Policy types exhibit varying claim and revenue patterns.
